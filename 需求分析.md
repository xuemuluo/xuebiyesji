## 1. 项目概述

### 1.1 项目背景

随着教育数字化转型的不断深化，中学学生学籍管理的规范化、高效化和精准化需求日益突出。传统中学学籍管理存在以下局限性：

| 问题类型 | 具体表现 |
|----------|----------|
| 人工录入繁琐 | 纸质档案管理效率低，信息录入重复劳动 |
| 数据分散混乱 | 多系统数据不互通，信息孤岛现象严重 |
| 查询统计耗时 | 手工统计报表耗时长，易出错 |
| 学籍变动追溯困难 | 变动记录不完整，历史数据难以追溯 |

构建一个安全可靠、操作便捷的学生学籍管理系统，在简化教务管理流程、保障学籍数据准确性、提升教育管理效率方面意义重大。

### 1.2 系统定位

**StuRegSys** 是面向中学的学生学籍管理系统，核心业务涵盖：

| 功能模块 | 功能范围 | 目标用户 |
|----------|----------|----------|
| 学生信息管理 | 学生基本信息录入、查询、修改、导出 | 教务管理员、班主任 |
| 学籍变动管理 | 转入、转出、休学、复学、毕业等变动处理 | 教务管理员 |
| 班级管理 | 班级信息维护、班主任分配 | 教务管理员 |
| 成绩管理 | 学生成绩录入、查询、统计 | 任课教师、班主任 |
| 考勤管理 | 学生考勤记录、统计 | 班主任、任课教师 |
| 消息管理 | 系统消息发送、接收 | 所有角色 |
| 统计分析 | 学籍数据统计、报表生成 | 教务管理员、班主任 |
| 系统管理 | 用户管理、操作日志 | 系统管理员 |

### 1.3 技术架构

| 层次 | 技术选型 | 版本 | 说明 |
|------|----------|------|------|
| 前端框架 | Vue 3 | 3.4.x | 响应式框架 |
| UI组件库 | Element Plus | 2.4.x | 企业级组件 |
| 后端框架 | Spring Boot | 3.1.x | MVC架构 |
| 持久层 | MyBatis Plus | 3.5.x | ORM框架 |
| 数据库 | MySQL | 8.0 | 主数据库 |
| 认证方案 | JWT Token | - | 无状态认证 |

### 1.4 系统特性

- **多终端访问**: 支持多终端浏览器访问，无需本地安装
- **操作直观**: 界面友好，操作简便，降低学习成本
- **数据安全**: 完善的权限控制和数据备份机制
- **追溯完整**: 学籍变动全程记录，支持历史追溯
- **多角色支持**: 5种角色，不同权限控制

| 教务管理员 | AcademicLayout | 学生管理、班级管理、学籍变动、报表统计 |

## 3. 用户角色分析

### 3.1 角色定义

| 角色标识 | 角色名称 | 角色描述 | 数量级 |
|----------|----------|----------|--------|
| ADMIN | 系统管理员 | 系统最高权限，负责系统配置和全局管理 | 少量 |
| ACADEMIC | 教务管理员 | 负责学籍管理、变动处理等核心业务 | 少量 |
| HEADTEACHER | 班主任 | 负责本班学生信息查看和管理 | 中等 |
| TEACHER | 任课教师 | 查看任教班级学生基本信息、录入成绩 | 中等 |
| PARENT | 家长 | 查看子女学籍信息、成绩、考勤 | 大量 |

### 3.2 角色详细说明

#### 3.2.1 系统管理员
- 负责系统初始化配置
- 管理所有用户账号
- 查看系统日志
- 全局学生、班级、学籍变动管理

#### 3.2.2 教务管理员
- 学生信息录入、修改、删除
- 学籍变动办理（转入、转出、休学、复学、毕业）
- 班级管理
- 统计报表导出

#### 3.2.3 班主任
- 查看本班学生信息
- 修改本班学生部分信息
- 查看本班学籍变动记录
- 管理本班学生考勤
- 查看本班学生成绩
- 查看学生档案
- 接收和发送消息

#### 3.2.4 任课教师
- 查看任教班级学生基本信息
- 录入和管理学生成绩
- 查看学生考勤
- 接收和发送消息

#### 3.2.5 家长
- 查看子女学籍信息
- 查看子女学籍变动记录
- 查看子女成绩
- 查看子女考勤
- 接收系统消息
- 无法修改任何信息

## 4. 功能模块详解

### 4.1 学生信息管理模块

#### 4.1.1 学生列表

**功能描述**: 查询和管理学生基本信息

**功能点**:
- 分页查询学生列表
- 多条件筛选（年级、班级、姓名、学籍号、状态）
- 查看学生详细信息
- 编辑学生信息
- 删除学生（逻辑删除）
- 导出学生列表（Excel）

**数据项**:
- 基本信息：姓名、性别、出生日期、民族、籍贯
- 学籍信息：学籍号、年级、班级、入学日期、学籍状态
- 联系信息：联系电话、家庭住址、家长姓名、家长电话
- 其他信息：身份证号、户口所在地、照片

#### 4.1.2 新增学生

**功能描述**: 录入新学生信息

**功能点**:
- 表单录入学生基本信息
- 学籍号自动生成或手动输入
- 照片上传
- 必填项校验
- 身份证号格式校验
- 学籍号唯一性校验

#### 4.1.3 批量导入

**功能描述**: 批量导入学生信息

**功能点**:
- 下载导入模板
- Excel文件上传
- 数据校验（格式、唯一性、必填项）
- 校验结果预览
- 错误数据导出
- 确认导入

#### 4.1.4 信息变更记录

**功能描述**: 记录学生信息修改历史

**功能点**:
- 查看学生信息变更历史
- 记录变更字段、原值、新值
- 记录变更人、变更时间
- 支持按学生、时间范围查询

### 4.2 学籍变动管理模块

#### 4.2.1 转入办理

**功能描述**: 处理学生转入业务

**业务流程**:
```
转入申请 → 资料审核 → 学籍建立 → 分班确认 → 办理完成
```

**功能点**:
- 录入转入学生信息
- 上传转入材料（转学证明、学籍证明等）
- 审核转入申请
- 建立学籍档案
- 分配班级
- 生成转入记录

#### 4.2.2 转出办理

**功能描述**: 处理学生转出业务

**业务流程**:
```
转出申请 → 审批确认 → 学籍转出 → 档案移交 → 办理完成
```

**功能点**:
- 选择转出学生
- 录入转出信息
- 上传转出材料
- 审批转出申请
- 生成转学证明
- 更新学籍状态
- 生成转出记录

#### 4.2.3 休学办理

**功能描述**: 处理学生休学业务

**业务流程**:
```
休学申请 → 材料审核 → 审批确认 → 学籍保留 → 办理完成
```

**功能点**:
- 选择休学学生
- 录入休学信息
- 上传休学材料（医院证明等）
- 审批休学申请
- 更新学籍状态
- 设置休学期限
- 生成休学记录

#### 4.2.4 复学办理

**功能描述**: 处理学生复学业务

**业务流程**:
```
复学申请 → 资格审核 → 审批确认 → 分班安排 → 办理完成
```

**功能点**:
- 查询休学学生
- 录入复学信息
- 上传复学材料
- 审批复学申请
- 分配班级
- 更新学籍状态
- 生成复学记录

#### 4.2.5 毕业办理

**功能描述**: 处理学生毕业业务

**业务流程**:
```
毕业申请 → 资格审核 → 毕业审批 → 学籍注销 → 档案移交 → 办理完成
```

**功能点**:
- 选择毕业学生
- 录入毕业信息
- 毕业资格审核
- 审批毕业申请
- 更新学籍状态为"已毕业"
- 生成毕业生档案
- 生成毕业证书编号

#### 4.2.6 变动记录查询

**功能描述**: 查询学籍变动历史记录

**功能点**:
- 分页查询变动记录
- 多条件筛选（学生姓名、变动类型、时间范围、状态）
- 查看变动详情
- 导出变动记录

#### 4.2.7 变动趋势分析

**功能描述**: 分析学籍变动趋势

**功能点**:
- 按时间周期统计（周、月、季、年）
- 变动数量趋势图
- 变动类型分布
- 数据导出
